# Tabuadas-Python

Este programa em Python gera tabuadas com intervalos personalizados, permitindo que o usuário escolha o início e o fim da tabuada, bem como o início e o fim do multiplicador. O programa também valida entradas para garantir que apenas números positivos sejam aceitos.

💡 **Como funciona**

1. O usuário informa o início e o fim da tabuada.  
2. O usuário define o início e o fim do multiplicador.  
3. O programa valida entradas, rejeitando números negativos ou não numéricos.  
4. As tabuadas são exibidas de forma organizada no console.  

---

🖥️ **Como testar**

1. Copie todo o código para um arquivo com extensão `.py`.  
2. Abra no terminal ou em uma IDE Python (ex.: VS Code, PyCharm).  
3. Execute o programa digitando:
   
   ```bash
    python Tabuada.py
> Certifique-se de ter o Python instalado e configurado no seu sistema.
5. Insira valores positivos quando solicitado.  
6. O programa exibirá as tabuadas correspondentes ao intervalo escolhido.  

> Experimente com valores como início da tabuada 2, fim 5 e multiplicadores de 1 a 10 para ver os resultados.
